# Микробная Колония 🧬

> **Файл:** `games/gridColony.js`  
> **Слоган:** Участники контролируют микробы на сетке, захватывают территории и расширяют свои колонии.

## Суть игры

Каждый участник — микроб на двумерной сетке. Микробы случайно перемещаются, захватывают пустые клетки, тратят энергию и вступают в конфликты на чужой территории. Выбывает тот, у кого закончилась энергия или территория. Побеждает последняя колония.

## Подготовка

### Сетка

| Параметр | Значение |
|----------|----------|
| Размер клетки | 20 px |
| Ширина / высота | Зависит от размера canvas |

### Стартовые позиции

Участники распределяются по **четырём углам** (циклически):

| Индекс % 4 | Угол |
|------------|------|
| 0 | Верхний левый (20% × 20%) |
| 1 | Верхний правый (80% × 20%) |
| 2 | Нижний левый (20% × 80%) |
| 3 | Нижний правый (80% × 80%) |

Каждый начинает с блока **3×3 клетки** (9 клеток территории) вокруг стартовой точки.

## Параметры микроба

| Параметр | Значение |
|----------|----------|
| Начальная территория | 9 клеток |
| Энергия | 100 / 200 max |
| Кулдаун движения | 0.5 сек |

## Движение

Каждые 0.5 сек микроб делает шаг в случайном направлении (dx, dy ∈ {−1, 0, 1}).

### Пустая клетка

- Клетка **захватывается**.
- Территория: **+1**.
- Энергия: **+5**.

### Своя клетка

- Просто перемещение, без изменений.

### Чужая клетка — конфликт

Сравнивается размер территории (`territory`):

| Условие | Результат |
|---------|-----------|
| Атакующий больше | Захват клетки, атакующий +1 территория, защитник −1 |
| Защитник больше | Атакующий −1 территория |
| Равны | Ничего не происходит (неявно — нет ветки в коде) |

Если территория падает до **0** — микроб **выбывает**.

## Расход энергии

- Пассивный расход: **5 единиц/сек**.
- При расширении (см. ниже): **−10** за клетку.

Если энергия ≤ **0** — микроб **выбывает**.

## Пассивное расширение

Каждый кадр у каждого живого микроба:

- **2% шанс** захватить случайную **соседнюю пустую** клетку.
- Условие: энергия > **50**.
- Стоимость: **−10 энергии**, **+1 территория**.

## Условия выбывания

| Причина | Условие |
|---------|---------|
| Голод | Энергия ≤ 0 |
| Уничтожение колонии | Территория ≤ 0 |

## Эскалация

При общей эскалации все живые микробы получают **+50 энергии** (не выше максимума 200). Это даёт им возможность агрессивнее расширяться.

## Визуальные индикаторы

- **Территория** — полупрозрачная заливка клеток цветом владельца.
- **Микроб** — круг с белым ядром и обводкой.
- **Полоска энергии** над микробом (зелёная / жёлтая / красная).
- **Число территории** под микробом.

## Динамика игры

Начальная фаза — мирный рост из углов. По мере сближения колоний начинаются конфликты, где решает накопленная территория. Микробы с малым запасом энергии уязвимы к пассивному расходу и не могут расширяться.
